## Overview

This repository is a comprehensive, production-oriented implementation of fundamental and advanced Data Structures and Algorithms, built entirely from scratch using **C++**. 

Rather than relying on the C++ Standard Template Library (STL), this project demonstrates a deep understanding of memory management, pointer manipulation, and Object-Oriented Programming (OOP). The codebase is highly modular, ensuring strict separation of concerns, and is heavily focused on algorithmic efficiency.

**Important Note on Repository Structure:** To maintain a clean and reviewable history, the codebase is meticulously divided into **Git Branches**. Each core topic (e.g., `LinkedList`, `Graphs`, `DynamicProgramming`) was developed in isolation on its own branch. Reviewers can easily navigate to specific branches to inspect isolated implementations before exploring the unified `main` branch.

---

## Architecture & Code Organization

The project follows a modular, object-oriented approach utilizing C++ `namespaces` to prevent naming collisions and ensure clean code integration. 

### 1️⃣ Core Data Structures
- **Arrays & Matrices:** Custom Array ADT, Sparse Matrices, Diagonal/Triangular Matrices, and Polynomial representations.
- **Linked Lists:** Singly, Doubly, and Circular Linked Lists.
- **Stacks & Queues:** Implementations via Arrays and Linked Lists. *Note: Queue implementations specifically highlight strict linear queue behaviors.*
- **Trees:** Binary Trees, Binary Search Trees (BST), AVL Trees, and Heap Data Structures.

### 2️⃣ Custom Graph Library
- Built a robust, independent Graph library from the ground up.
- Representations using both Adjacency Matrix and Adjacency Lists.
- **Traversal:** Breadth-First Search (BFS) and Depth-First Search (DFS).
- **Shortest Path:** Dijkstra’s Algorithm.
- **Minimum Spanning Tree (MST):** Kruskal’s Algorithm (using Disjoint Sets) and Prim’s Algorithm.

### 3️⃣ Algorithm Strategies
- **Sorting & Searching:** Linear/Binary Search, Quick Sort, Merge Sort, etc.
- **Hashing Techniques:** Chaining, Linear Probing, Quadratic Probing, Double Hashing.
- **Divide and Conquer:** Optimized approaches for recursive problems.
- **Greedy Method:** Knapsack, Optimal Merge Patterns.
- **Dynamic Programming (DP):** 0/1 Knapsack, Matrix Chain Multiplication, Longest Common Subsequence (LCS), Maximum Subarray Sum.
- **Backtracking:** Rat in a Maze, N-Queens Problem, Sudoku Solver, Permutation bounding.

## Key Features

- **Built From Scratch:** Manual memory allocation and raw pointer manipulation.
- **Branch-Driven Development:** Topic-specific branches for clean code isolation and easy code-review.
- **Object-Oriented Design:** Extensive use of classes, templates, and operator overloading.
- **Namespace Isolation:** Preventing structural conflicts across hundreds of files.
- **Memory Safety:** Implementation of proper destructors and cleanup methods to prevent memory leaks.
- **Algorithmic Complexity Optimization:** Focus on writing time and space-efficient code.
